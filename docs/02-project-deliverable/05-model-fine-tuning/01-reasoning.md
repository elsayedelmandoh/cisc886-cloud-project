## training configuration

- **Dataset**: S3 processed data (from Section 4 - EMR preprocessing)
  - Location: `s3://25xrvl-s3/processed/train/` and `s3://25xrvl-s3/processed/val/`
  - Languages: Python, JavaScript, Go
- **Training samples**: from S3 processed train split
- **Validation samples**: from S3 processed val split
- **Trainable parameters**: ~0.1% of total (LoRA adapters only)

## colab setup

Before running on Google Colab, set up credentials:

1. In Colab, add a secret with AWS credentials:
   - `AWS_ACCESS_KEY_ID`: from .env
   - `AWS_SECRET_ACCESS_KEY`: from .env
   - `REGION`: us-east-1

2. Or set environment variables in first cell:
   ```python
   import os
   os.environ["AWS_ACCESS_KEY_ID"] = "your-key"
   os.environ["AWS_SECRET_ACCESS_KEY"] = "your-secret"
   os.environ["HF_TOKEN"] = "your-huggingface-token"
   ```
